FunFunc

A functional js lib

Installation

To use with node:

                    
                        npm i funfunc

Then in code:

                    
                        const F = require('funfunc')

Via CDN in HTML:

                    
                        <script src="https://cdn.jsdelivr.net/npm/funfunc@0.0.1/dist/funfunc.min.js"></script>

Then in code without import:

                    
                        const curried = F.curry(add)

Or in module:

                    
                        import F from 'funfunc'
